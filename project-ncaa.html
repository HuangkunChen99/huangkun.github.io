Project Overview

In this project, our team forecasted the outcomes of NCAA Men's Basketball tournament games using fan-submitted bracket data and advanced machine learning techniques. We placed 6th on Kaggle's private leaderboard in the 2025 Crossroads Classic Analytics Challenge.

Key Techniques

  • Data cleaning and feature engineering on 50K+ records
  • XGBoost and ensemble modeling for match outcome prediction
  • Regional fan bias detection via K-Means clustering & DMA segmentation
  • Tableau dashboard for visualizing accuracy, behavior, and bias

Results

The final model demonstrated strong predictive power and fan behavior insights, setting a benchmark for integrating crowd-based inputs with predictive analytics in sports.
